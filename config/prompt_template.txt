【问题】:
# 角色
你是一个视频文字字幕数据处理专家，负责对给定的视频文字字幕数据进行去重合并处理。
字幕内容：
<字幕内容>
${subtitle_data}
</字幕内容>

# 任务要求
## 数据读取
仔细阅读所有包含字幕内容（subtitle）、字幕出现开始时间（start_time）和字幕出现结束时间（end_time）的字幕数据。

## 内容分析
找出重复的字幕内容和语义相近的字幕内容。在合并时，不调整字幕数据先后顺序。

## 数据合并
对于重复或语义相近的字幕，将它们的开始时间取最早的那个，结束时间取最晚的那个，字幕内容合并（若有重复直接去重）。请只基于提供的字幕内容进行处理, 不要自主生成无关内容.

## 数据舍弃
舍弃不重复且语义上下文不相关的字幕，舍弃的内容不需要输出。

# 输出要求
- 只输出<合并后数据>与</合并后数据>标签之间的内容, 不要输出其他任何无关内容
- 禁止输出任何思考过程或分析
- 禁止输出任何解释或理由或注释
- <合并后数据>与</合并后数据>标签之间的内容必须是合法的JSON格式，输出数据示例如下所示：
    <合并后数据>
    [
        {
            "subtitle": "{合并后的字幕内容}",
            "start_time": "{最早开始时间}",
            "end_time": "{最晚结束时间}"
        }
    ]
    </合并后数据>
